## 📝 使用说明

### 开发环境启动
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

### 环境配置
1. 复制 `env.example` 为 `.env.local`
2. 根据实际情况配置API地址和其他参数
3. 启动后端API服务（当前使用模拟数据）

### 部署建议
- 使用 `npm run build` 构建生产版本
- 部署 `dist` 目录到web服务器
- 配置API代理和环境变量
- 确保图片资源正确上传
